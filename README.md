# igarin14pm-typescript-utils

TypeScript の自作ユーティリティ関数をまとめたリポジトリです。

# ファイル

## scripts/src/timers-promises.ts

`setTimeout()`/`setInterval()` をブラウザで `async`/`await` を使って呼び出せるように、 `Promise` でラップした関数を定義。
